# Obrázek cert-testimonial-maxhair.png

## Problém
Fotografie spokojeného klienta v sekci "O nás" (cert sekce) není na stránce vidět, protože soubor `cert-testimonial-maxhair.png` nebyl nahrán na WordPress.

## Umístění v kódu
- **Soubor:** `pages/o-nas.json`
- **Sekce:** O nás Cert (Garance kvality, Záruční certifikát)
- **URL:** `https://mediumseagreen-gaur-406121.hostingersite.com/wp-content/uploads/2026/02/cert-testimonial-maxhair.png`

## Jak obrázek přidat

### Možnost 1: Přes WordPress admin (doporučeno)
1. Přihlaste se do WordPress adminu
2. **Média** → **Přidat nové**
3. Nahrajte soubor `cert-testimonial-maxhair.png`
4. WordPress ho uloží do `wp-content/uploads/2026/02/`

### Možnost 2: Lokální soubor + nasazení
1. Vložte soubor `cert-testimonial-maxhair.png` do složky `assets/images/` v projektu
2. Zkopírujte do: `wp-content/uploads/2026/02/cert-testimonial-maxhair.png`
3. Nahrajte složku `wp-content/uploads/2026/` na server přes FTP/SFTP nebo Hostinger File Manager

### Možnost 3: Přes Hostinger File Manager
1. Přihlaste se do Hostinger
2. File Manager → `public_html/wp-content/uploads/`
3. Vytvořte složky `2026/02/` (pokud neexistují)
4. Nahrajte soubor `cert-testimonial-maxhair.png`
