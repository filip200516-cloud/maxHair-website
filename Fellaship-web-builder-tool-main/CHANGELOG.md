# Changelog - Fellaship Web Builder Tool

## [1.0.0] - 2025-01-02

### Přidáno
- **CURSOR-AI-GUIDE.md** - Kompletní průvodce pro Cursor AI
- **.cursorrules** - Pravidla a instrukce pro Cursor AI
- **QUICK-START-AI.md** - Rychlý start pro AI
- **AI-CONTEXT.md** - Klíčové informace pro AI
- Podpora pro konfigurovatelné názvy templates v `config.json` → `templateNames`

### Změněno
- Projekt přejmenován na **Fellaship Web Builder Tool**
- Odstraněny všechny hardcoded reference na konkrétní projekty (MaxHair)
- `config.json.example` aktualizován na generické hodnoty
- Dokumentace přepracována pro univerzální použití
- `sync.js` aktualizován pro použití konfigurovatelných názvů templates

### Odstraněno
- Hardcoded reference na MaxHair projekt
- Specifické cesty a názvy souborů
- Projekt-specifické příklady

### Důležité
- Tool je nyní **univerzální** - funguje s jakýmkoliv WordPress + Bricks projektem
- Všechna konfigurace je v `config.json` - žádné hardcoded hodnoty
- Tool je navržen pro použití **pouze přes Cursor AI**
